*clang-format*

Plugin to run clang-format on the current buffer.

Prerequisites:
1. A .clang-format or _clang-format file must be in the directory of the
   current file, or one of its ancestors.
2. Your PATH environment variable must contain the path to the clang-format
   executable.
3. Filetype plugins must be enabled in Vim. Use :filetype? to check if
   'plugin=on'. Add 'filetype plugin on' to your vimrc to force filetype
   plugins to be on.

To use the plugin, you can:
1. In normal mode, type *<Leader>f* (see |mapleader|),
2. In normal mode, use the command *:Format* , or
3. In the GUI, use the menu Plugin->C++->Clang Format.

The plugin will run clang-format on the current file, in place. If any changes
are made by clang-format, Vim should ask if you want to load the changes.
